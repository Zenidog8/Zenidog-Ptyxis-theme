# Zenidog-Ptyxis-theme
A theme for ptyxis terminal

## Installation
1. Download the repository
   > ```bash git clone https://github.com/Zenidog8/Zenidog-Ptyxis-theme.git && cd Zenidog-Ptyxis-theme```
1. Copy the file to the palettes folder
   > ```bash cp Zenidog.palette ~/.local/share/org.gnome.Ptyxis/palettes/```
1. Delete the repository
   > ```bash cd .. && rm -rf Zenidog-Ptyxis-theme```
1. Set it on the terminal configuration panel
